Matrices and Operations

Matrix Definition

A matrix is a rectangular table of numbers arranged in m rows and n columns. It is denoted as:

a11 a12 ⋯ a1n
a21 a22 ⋯ a2n
⋮ ⋮ ⋱ ⋮
am1 am2 ⋯ amn

Special Matrices

Square Matrix: Same number of rows and columns (m = n)

Identity Matrix: Diagonal elements are 1, others are 0

1 0 0
0 1 0
0 0 1

Triangular Matrix: All elements below (or above) the diagonal are zero

Zero Matrix: All elements are zero

Matrix Operations

Addition: Matrices must be same size

a11+b11 a12+b12
a21+b21 a22+b22

Scalar Multiplication:

k × A = [k·aij]

Matrix Multiplication: Columns of A must match rows of B

Cij = Σ(aik·bkj)

Determinants

Definition

A determinant is a scalar value that can be computed from a square matrix. Denoted as |A| or det(A).

For a 2×2 matrix:

a11 a12
a21 a22

|A| = a11a22 - a12a21

Example:

1 -3
-5 10

|A| = (1)(10) - (-3)(-5) = 10 - 15 = -5

3×3 Determinant

Using Sarrus' rule:

|A| = a11a22a33 + a21a32a13 + a12a23a31 - a13a22a31 - a23a32a11 - a21a12a33

Example:

1 2 3
-1 -1 -4
1 2 7

|A| = (1)(-1)(7) + (-1)(2)(3) + (2)(-4)(1) - (3)(-1)(1) - (2)(-4)(1) - (-1)(2)(7) = -7 -6 -8 +3 +8 +14 = 4

Minors and Cofactors

Minor (Mij): Determinant obtained by deleting the i-th row and j-th column.

Cofactor (Aij): Aij = (-1)i+j Mij

Example for a 3×3 matrix:

M11 = determinant of the submatrix without row 1 and column 1

Matrix Rank

Definition

The rank of a matrix is the highest order of a non-zero minor that can be obtained from the matrix.

Denoted as r(A) or rank(A).

Elementary Operations

Elementary operations that preserve rank:

  1. Swapping two rows or columns
  2. Multiplying a row or column by a non-zero scalar
  3. Adding a multiple of one row to another row

Examples

Matrix A:

2 2 3
1 1 2

Rank(A) = 2

Inverse Matrices

Definition

A matrix B is the inverse of matrix A if:

AB = BA = I (identity matrix)

Denoted as A-1

Calculation

For a square matrix A with |A| ≠ 0:

A-1 = (1/|A|) · adj(A)

Where adj(A) is the adjugate matrix (transpose of the cofactor matrix)

Regular vs. Singular

Regular Matrix: Determinant ≠ 0, has an inverse

Singular Matrix: Determinant = 0, no inverse exists

Linear Systems

Definition

A system of linear equations:

a11x1 + a12x2 + ... + a1nxn = b1

a21x1 + a22x2 + ... + a2nxn = b2

...

am1x1 + am2x2 + ... + amnxn = bm

Matrix Form

A · X = B

Where:

A = coefficient matrix

X = variable vector

B = constant vector

Kronecker-Capelli Theorem

A system has a solution if and only if:

rank(A) = rank([A|B])

Where [A|B] is the augmented matrix

Gaussian Elimination

Method Overview

A method for solving systems of linear equations by transforming the augmented matrix to row echelon form through elementary row operations.

Elementary Operations

Operations that preserve solution set:

  1. Multiply an equation by a non-zero constant
  2. Add a multiple of one equation to another
  3. Swap two equations

Solution Cases

No solution: If any row has [0 0 ... 0 | c] with c ≠ 0

Unique solution: If rank = number of variables

Infinitely many solutions: If rank < number of variables
